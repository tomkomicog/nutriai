# NutriAI

NutriAI is an AI-powered nutrition and fitness platform that helps users plan meals, calculate calories and generate workout programs.

## Features

- AI meal plan generator
- AI workout generator
- calorie and macro tracking
- food recognition from photos
- recovery and fatigue analysis
- nutrition calculators (TDEE, BMR, BMI, macros)
- 1300+ nutrition guides and tools
